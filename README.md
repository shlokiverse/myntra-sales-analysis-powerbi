# MYNTRA Sales Analysis Report

## Overview
This project presents a Power BI dashboard analyzing Myntra sales data to extract business insights and support decision-making.

## Key Insights
- Revenue and order trends
- Category-wise sales performance
- Regional distribution
- KPI tracking

## Tools Used
- Power BI
- Data Cleaning & Transformation
- DAX Calculations
